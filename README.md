# PRODIGY_ML_02
# Customer Segmentation using K-Means Clustering

## 📌 Objective
Develop a K-Means Clustering model to group retail store customers based on their purchasing behavior using their Annual Income and Spending Score.

## 📂 Dataset
- **Dataset:** Mall Customers Dataset
- **Source:** Kaggle

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## 📊 Features Used
- Annual Income (k$)
- Spending Score (1-100)

## 🤖 Machine Learning Algorithm
- K-Means Clustering

## 📈 Project Workflow
- Import and explore the dataset
- Perform Exploratory Data Analysis (EDA)
- Select relevant features
- Scale the data using StandardScaler
- Determine the optimal number of clusters using the Elbow Method
- Train the K-Means model
- Visualize customer segments
- Interpret the clusters

## 📌 Results
The Elbow Method identified **5** as the optimal number of clusters. The model successfully segmented customers into different groups based on their income and spending behavior, helping identify customer categories such as:
- High Income – High Spending
- High Income – Low Spending
- Low Income – High Spending
- Low Income – Low Spending
- Average Customers

## 🎯 Outcome
This project demonstrates how unsupervised machine learning can be used for customer segmentation, enabling businesses to create targeted marketing strategies and improve customer engagement.
